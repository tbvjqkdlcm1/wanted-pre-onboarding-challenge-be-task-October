(3) 웹 브라우저에 네이버 를 검색하고 화면에 네이버 화면이 출력이 될 때 까지 내부적으로 어떤 동작들이 수행이 되는지 설명해주세요.

1. 웹 브라우저에 네이버를 검색하면
2. DNS가 네이버 URL 주소를 컴퓨터가 읽을 수 있는 주소로 변환한 후 DNS 서버에서 찾아서 네이버 URL 정보와 함께 전달합니다.
3. 네이버 URL 정보와 전달받은 IP 주소는 HTTP 프로토콜을 사용하여 HTTP request 메시지를 생성하고
4. 이렇게 생성된 HTTP request 메시지는 TCP 프로토콜을 사용하여 인터넷을 거쳐 해당 IP 주소의 컴퓨터로 전송되고 HTTP 프로토콜을 사용하여 네이버 URL 정보로 변환됩니다.
5. 웹 서버는 도착한 네이버 URL 정보에 해당하는 데이터를 검색하고 검색된 웹 페이지 데이터는 또다시 HTTP 프로토콜을 사용하여 HTTP response 메시지를 생성합니다.
6. 이렇게 생성된 HTTP response 메시지는 TCP 프로토콜을 사용하여 인터넷을 거쳐 원래 클라이언트로 전송되고
7. 도착한 HTTP response 메시지는 HTTP 프로토콜을 사용하여 웹 페이지 데이터로 변환됩니다.
8. 유저 눈에 볼 수 있게 뿌립니다.
